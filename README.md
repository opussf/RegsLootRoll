# RegsLootRoll

WoW addon to facilitate raid loot rolls

## What to do.

* Notify those in raid about the loot rules.
* Notify new players to the raid the loot rules.
* Track :NEED: uses
** Warn if they are used many times
** Warn if they are posted after the /roll
